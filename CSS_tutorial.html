Čo je CSS?


  • CSS je skratka pre Kaskádové štýly
  • CSS popisuje, ako sa majú HTML elementy zobraziť na obrazovke.
  • CSS šetrí veľa práce. Dokáže controlovať rozložnie viacerých webových stránok naraz.
  • Externé šablóny so štýlmi sú uložené v súboroch .css

Prečo používať CSS?

  • CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Priklad

body {
    background-color: lightblue;
}

h1 {
   color: white;
   text-align: center;
}
p {
   font-family: verdana;
   font-size: 20px;
}





CSS vyriešilo veľký problém


  • HTML NIKDY nemalo obsahovať značky na formátovanie webovej stránky!
  • HTML was created to describe the content of a web page, like:

Priklad

<h1> Toto je nadpis </h1>
<p> Toto je paragraf </p>


  • Keď boli do špecifikácie HTML 3.2 pridané značky ako font a atribúty farieb, pre vývojárov webu to začalo nočnú moru. Vývoj veľkých webových stránok, kde sa na každú stránku pridávali fonty a informácie o farbách, sa stal dlhým a drahým procesom.
  • Na vyriešenie tohto problému vytvorilo konzorcium World Wide Web CSS.
  • CSS odstránilo formátovanie štýlu zo stránky HTML!






CSS syntax


  • Selektor udáva ktorý HTML element chceme štýlovať.
  • Deklaračný blok obsahuje jednu alebo viac deklarácií oddelených bodkočiarkou.
  • Každá deklarácia obsahuje názov vlastnosti CSS a hodnotu oddelenú dvojbodkou.
  • Viaceré deklarácie CSS sú oddelené bodkočiarkami a bloky deklarácií sú ohraničené zloženými zátvorkami.


Priklad

p {
    color : red;
    text-align: center;
}



Vysvetlenie prikladu:

 p 
je selektor v CSS (ukazuje na prvok HTML, ktorý chcete upraviť: <p>).
 color 
je vlastnosť a červená je hodnota vlastnosti
 text-align 
je vlastnosť a stred je hodnota vlastnosti






CSS zoznamy


V HTML sú dva hlavné druhy listov:

  • Neusporiadany list <ul> - jednotlivé zložky v liste sú označené s odrážkami
  • Usporiadany list <ol> - jednotlivé zložky v liste sú označené číslom alebo s písmenkom

V CSS vieme nastaviť vlastnosti na:

  • Nastaviť iný typ odrážky v usporiadanom liste
  • Nastaviť iný typ odrážky v neusporiadanom liste
  • Nastaviť obrázok ako odrážku
  • Pridať farbu pozadia listu aj jednotlivým zložkám

Priklad 1

ol{
    list-style-type: decimal
}

ol{

}

  3. text

ul{
    list-style-type: circle
}

CSS Gradienty


  • CSS gradienty ti dovolia hladký prechod medzi dvoma alebo viacerými farbami
  • CSS má tri typy gradientov:
    1. Lineárny Gradient
    2. Radiálny Gradient
    3. Kužeľový Gradient

CSS Lineárne Gradienty


Syntax príkazu:


background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

Smer Gradientu je defaultne nastavený na spodok


div {
    background-image: linear-gradient(red, yellow);
}



Smer Gradientu - do prava


div {
    background-image: linear-gradient(to right, red, yellow);
}



Smer Gradientu - Diagonala


div {
    background-image: linear-gradient(to bottom right, red, yellow);
}



Smer Gradientu - Konkrétny úhol


div {
    background-image: linear-gradient(200deg, red, yellow);
}







CSS Tiene


  • S CSS vieme pridať tiene textu alebo elementom
  • V našom prípade budeme pracovať s vlastnosťami:
    1. text-shadow
    2. box-shadow

CSS Tiene Pri Texte

  • V CSS vlastnosť text-shadow pridáva tieň textu
h1 {
    text-shadow: 3px 3px red;
}

   Text Shadow

h1 {
    text-shadow: 3px 3px 3px red;
}

   Text Shadow



CSS Tiene Pri Elemente

  • V CSS vlastnosť box-shadow pridáva tieň elementu
div {
    box-shadow: 10px 10px blue;
}



CSS Rozostrené Tiene Pri Elemente

  • V CSS vlastnosť box-shadow pridáva tieň elementu
div {
    box-shadow: 10px 10px 20px blue;
}
