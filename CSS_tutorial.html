Čo je CSS?

  • CSS je skratka pre Kaskádové štýly
  • CSS popisuje, ako sa majú HTML elementy zobraziť na obrazovke.
  • CSS šetrí veľa práce. Dokáže controlovať rozložnie viacerých webových stránok naraz.
  • Externé šablóny so štýlmi sú uložené v súboroch .css

Prečo používať CSS?

  • CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Priklad

body {
    background-color: lightblue;
}

h1 {
   color: white;
   text-align: center;
}
p {
   font-family: verdana;
   font-size: 20px;
}





CSS vyriešilo veľký problém

  • HTML NIKDY nemalo obsahovať značky na formátovanie webovej stránky!
  • HTML was created to describe the content of a web page, like:

Priklad

<h1> Toto je nadpis </h1>
<p> Toto je paragraf </p>


  • Keď boli do špecifikácie HTML 3.2 pridané značky ako font a atribúty farieb, pre vývojárov webu to začalo nočnú moru. Vývoj veľkých webových stránok, kde sa na každú stránku pridávali fonty a informácie o farbách, sa stal dlhým a drahým procesom.
  • Na vyriešenie tohto problému vytvorilo konzorcium World Wide Web CSS.
  • CSS odstránilo formátovanie štýlu zo stránky HTML!






CSS syntax

  • Selektor udáva ktorý HTML element chceme štýlovať.
  • Deklaračný blok obsahuje jednu alebo viac deklarácií oddelených bodkočiarkou.
  • Každá deklarácia obsahuje názov vlastnosti CSS a hodnotu oddelenú dvojbodkou.
  • Viaceré deklarácie CSS sú oddelené bodkočiarkami a bloky deklarácií sú ohraničené zloženými zátvorkami.


Priklad

p {
    color : red;
    text-align: center;
}



Vysvetlenie prikladu:

 p 
je selektor v CSS (ukazuje na prvok HTML, ktorý chcete upraviť: <p>).
 color 
je vlastnosť a červená je hodnota vlastnosti
 text-align 
je vlastnosť a stred je hodnota vlastnosti
