Čo je HTML?

  • je skratka pre "Hyper Text Markup Language"
  • je základným jazykom na výrobu internetových stránok
  • popisuje štruktúru webovej stránky
  • sa skladá zo série elementov
  • elementy popisujú ako prehliadavač zobrazí kontent stránky

Priklad

<!DOCTYPE html>
<html>
<title>Názov stránky</title>
<body>

<h1>Môj prvý nadpis</h1>
<p>Môj prvý paragraf</p>

</body>
</html>

Vysvetlenie prikladu:

<!DOCTYPE html>
deklarácia definuje ,že tento dokument je verzie HTML5
<html>
je koreňom HTML stránky
<head>
je element ktorý obsahuje informácie o HTML stránke
<title>
je element ktorý špecifikuje názov HTML stránky (ktorý sa zobrazí v ldtište prehliadača)
<body>
je element ktorý definuje telo dokumentu a je to kontajner pre všetok viditelný kontent stránky
<h1>
je element ktorý definuje nádpis
<p>
je element ktorý definuje paragraf






HTML dokument

  • Všetky HTML dokumenty musia začínať s deklaráciou typu dokumentu: <!DOCTYPE html>
  • HTML dokument samotny začína s <html> a končí s </html>
  • Viditeľná časť HTML dokumentu je medzi <body> a </body>

Priklad

<!DOCTYPE html>
<html>
<title>Názov stránky</title>
<body>

<h1>Môj prvý nadpis</h1>
<p>Môj prvý paragraf</p>

</body>
</html>







deklarácia <!DOCTYPE html>

  • <!DOCTYPE> reprezentuje typ dokumentu a pomáha prehliadavačom zobraziť stránku správne
  • <!DOCTYPE> sa smie použiť iba raz, a to na začiatku, pred hocijakým HTML tagom
  • <!DOCTYPE> nieje citlivý na veľké a malé písmenká
  • deklarácia <!DOCTYPE> pre html 5 je:

<!DOCTYPE html>







HTML nadpisy

  • HTML nadpisy sú definované tagmi <h1> až <h6>
  • <h1> definuje najviac dôležitý nadpis. <h6> definuje najmenej dôležitý nadpis

Priklad

<h1>Toto je nadpis 1</h1>
<h2>Toto je nadpis 2</h2>
<h3>Toto je nadpis 3</h3>







HTML paragrafy

  • HTML paragrafy sú definované <p>:

Priklad

<p>Toto je praragraf.</p>
<p>Toto je praragraf.</p>







HTML linky

  • HTML linky sú definované tagom <a>:

Priklad

<a href= "https://www.google.com/"> Toto je link</a>







HTML obrázky

  • Obrázky v HTML sú definované tagom <img>
  • <img> obsahuje atribúty: zdrojový súbbor (src), alternatívny text (alt), width, height

Priklad

<img src="obrazok.png" alt="obrazok" width="50" height="80">
